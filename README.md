# Etherpass UI
Etherpass all-in-one web and desktop application.

## Getting started

### Prerequisites
- Vagrant
- VirtualBox

### Setting up VM for development
```bash
$ vagrant up
$ vagrant ssh
$ cd /vagrant # TODO auto cd; rename working directory
```

### NPM workflow:
- `npm install`
- `npm start`

Visit http://localhost:5577/
